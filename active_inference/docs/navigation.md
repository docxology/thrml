## Learning Paths

### Path 1: Complete Beginner

**Goal**: Understand active inference and run your first agent

1. ‚è±Ô∏è 5 min: Read [Getting Started: Core Concepts](getting_started.md#core-concepts)
2. ‚è±Ô∏è 10 min: Read [Theory: Overview](theory.md#overview)
3. ‚è±Ô∏è 15 min: Run [Example 01: Basic Inference](../examples/01_basic_inference.py)
4. ‚è±Ô∏è 20 min: Run [Example 02: Grid World Agent](../examples/02_grid_world_agent.py)
5. ‚è±Ô∏è 10 min: Review [Workflows: First Agent](workflows_patterns.md#creating-your-first-agent)

**Total Time**: ~1 hour

**Next**: Path 2 or start building custom agents

---

### Path 2: Practical Developer

**Goal**: Build and deploy active inference agents

1. ‚è±Ô∏è 10 min: Review [Architecture Overview](architecture.md#system-architecture)
2. ‚è±Ô∏è 15 min: Study [Core Module](module_core.md)
3. ‚è±Ô∏è 15 min: Study [Agent Module](module_agents.md)
4. ‚è±Ô∏è 20 min: Follow [Workflows: Training Loop](workflows_patterns.md#training-loop-pattern)
5. ‚è±Ô∏è 30 min: Build custom agent using patterns

**Total Time**: ~1.5 hours

**Next**: Path 3 for optimization or Path 4 for THRML

---

### Path 3: Performance Engineer

**Goal**: Optimize and scale active inference systems

1. ‚è±Ô∏è 10 min: Read [Performance Guide](performance.md)
2. ‚è±Ô∏è 15 min: Review [Workflows: Optimization](workflows_patterns.md#optimization-patterns)
3. ‚è±Ô∏è 10 min: Study [Resource Tracking](module_utils.md#resource-tracking)
4. ‚è±Ô∏è 20 min: Implement caching and JIT compilation
5. ‚è±Ô∏è 15 min: Profile with [Resource Tracker](workflows_patterns.md#memory-optimization)

**Total Time**: ~1-2 hours

**Next**: Advanced optimization techniques

---

### Path 4: THRML Integration Specialist

**Goal**: Master THRML integration for advanced inference

1. ‚è±Ô∏è 20 min: Read [THRML Integration Overview](thrml_integration.md#overview)
2. ‚è±Ô∏è 30 min: Study [THRML Components Reference](thrml_integration.md#thrml-components-reference)
3. ‚è±Ô∏è 15 min: Review [Integration Patterns](thrml_integration.md#integration-patterns)
4. ‚è±Ô∏è 45 min: Run [Example 11: THRML Comprehensive](../examples/11_thrml_comprehensive.py)
5. ‚è±Ô∏è 30 min: Build custom THRML factors

**Total Time**: ~2-3 hours

**Next**: Advanced THRML sampling and hardware acceleration

---

### Path 5: Researcher

**Goal**: Deep understanding of theory and implementation

1. ‚è±Ô∏è 30 min: Read [Complete Theory Guide](theory.md)
2. ‚è±Ô∏è 45 min: Study all [Module Documentation](module_index.md)
3. ‚è±Ô∏è 30 min: Review [Architecture in Detail](architecture.md)
4. ‚è±Ô∏è 60 min: Run all examples and analyze code
5. ‚è±Ô∏è 45 min: Read [Statistical Analysis](analysis_validation.md)

**Total Time**: ~3-4 hours

**Next**: Contribute to library or build novel applications
